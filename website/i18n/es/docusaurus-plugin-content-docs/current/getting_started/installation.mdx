Hay algunas formas de instalar FVM, y estamos trabajando en otras formas de hacerlo aún más simple.

## Independiente (Recomendado)

Puede descargar los paquetes independientes desde el repositorio de GitHub [aquí](https://github.com/leoafarias/fvm/releases).

Si usas el [Homebrew package manager](https://brew.sh) para Mac OS X, puedes instalar FVM ejecutando

brew tap leoafarias/fvm

brew uninstall fvm

## Paquete de Pub

También puede instalar FVM como un [paquete pub](https://pub.dev/packages/fvm).

Sin embargo, si planea usar FVM para administrar su [instalación global de Flutter](/docs/guides/global_version), le recomendamos que lo instale de forma independiente.

```bash
dart pub global activate fvm
```
